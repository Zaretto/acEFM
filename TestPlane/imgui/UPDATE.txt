How to update
-------------
1. Download the desired release zip from https://github.com/ocornut/imgui/releases
   or clone the repo and check out the tag you want.

2. Overwrite all files listed above from the new source tree.
   The backends/ files live under imgui/backends/ in the ImGui repo.

3. Check the IMGUI_VERSION string in imgui.h and update the version line
   at the top of this file.

4. If imconfig.h has local customisations, merge carefully rather than
   overwriting blindly.

5. Build TestPlane and fix any breaking API changes noted in imgui/docs/CHANGELOG.txt.
